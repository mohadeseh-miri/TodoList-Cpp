# Todo List (C++)

## معرفی پروژه
این پروژه در قالب درس **برنامه‌سازی پیشرفته** و با هدف تمرین مفاهیم اصلی
برنامه‌نویسی شیءگرا در زبان ++C انجام شده است.  
این برنامه یک Todo List ساده را پیاده‌سازی می‌کند که به کاربر اجازه می‌دهد
کارهای روزانه‌ی خود را مدیریت کند.

---

## هدف پروژه
هدف اصلی این پروژه آشنایی با مفاهیم زیر است:
- طراحی کلاس و استفاده از شیء
- کار با توابع و منوی متنی
- استفاده از اشاره‌گرها
- پیاده‌سازی مفاهیم شیءگرایی مانند وراثت و چندریختی
- مرتب‌سازی و فیلتر کردن داده‌ها

---

## توضیح کلی برنامه
در این پروژه یک برنامه کنسولی طراحی شده است که به کاربر اجازه می‌دهد
کارهای خود را ثبت، مشاهده و مدیریت کند.  
هر کار دارای اطلاعات مشخصی است و سیستم امکان مدیریت مجموعه‌ای از کارها
را فراهم می‌کند.

---

## امکانات برنامه
- افزودن تسک جدید
- نمایش تمام تسک‌ها
- نمایش تسک‌های انجام‌شده
- نمایش تسک‌های منقضی‌شده
- تغییر وضعیت تسک
- حذف تسک

---

## توضیح بخش‌های مهم کد

### نمایش تسک‌ها
برای نمایش تسک‌ها از تابع `display` استفاده شده است.  
نمایش همه‌ی تسک‌ها، تسک‌های منقضی‌شده و تسک‌های انجام‌شده
با استفاده از فیلترهای مختلف انجام شده است تا منطق نمایش
از منطق داده جدا باشد.

### مدیریت وضعیت تسک
برای مشخص کردن انجام‌شدن یا انجام‌نشدن تسک‌ها از متغیر وضعیت (`status`)
و توابع مربوط به آن استفاده شده است.  
این کار باعث خوانایی بیشتر کد و مدیریت بهتر تسک‌ها شده است.

### مرتب‌سازی بر اساس اولویت
مرتب‌سازی تسک‌ها با استفاده از الگوریتم ساده **Bubble Sort** انجام شده است.  
در این بخش فقط اشاره‌گرها جابه‌جا می‌شوند و داده‌ها کپی نمی‌شوند
که باعث ساده بودن و خوانایی کد می‌شود.
